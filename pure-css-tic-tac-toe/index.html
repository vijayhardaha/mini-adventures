Pure CSS Tic Tac Toe

A Tic-Tac-Toe game purely built on HTML & CSS. A Fun & Cool project to inspire some new developers so that they can learn new things and challenge themselves to build some complex things.

View on Github Share on Twitter

How to play?

  • Use "Player A" & "Player B" button to change player turns.
  • Use "Restart Game" button reset the game.
  • Checkmark is for "Player A" & Crossmark is for "Player B".
  • Note: After a player's move don't forget to switch the player turn.
  • Enjoy !
